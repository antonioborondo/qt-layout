# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2025-11-16

### Changed

- The long command-line option to load layouts is now named "restore".
- The short command-line option to list layouts is now "l".

## [1.2.0] - 2025-11-16

### Added

- Include the status bar layout when saving and loading.

## [1.1.0] - 2025-11-15

### Added

- Add a command-line option to list previously saved layouts.

## [1.0.0] - 2025-11-15

### Added

- Add a command-line option to save the layout of the left and bottom panels.
- Add a command-line option to load previously saved layouts.
